Overview

KFDataTableViewController is a generic controller base that manages a table view from an NSFetchRequest.

It will automatically insert or update cells when changes have been made to the NSFetchRequest.

Parameters

A table-view object requesting the cell.

reuseIdentifier

The reuse identifier we are requesting a cell for
