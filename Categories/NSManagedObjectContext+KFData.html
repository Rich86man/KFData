performWriteBlock:success:failure:

Asyncronously perform a block on the managed object context, then run a nested save.

- (void)performWriteBlock:(void ( ^ ) ( void ))writeBlock success:(void ( ^ ) ( void ))success failure:(void ( ^ ) ( NSError *error ))failure

Parameters

writeBlock

The block to run on the managed object context.

success

A block to run when the write block has been executed, and the managed object context has saved up to it’s parent.

failure

A block to run when there was a failure to save in the current block, or it’s parents.

Discussion

Once the block has been executed, it will run either the success or the failure block.
